# Real-World Video Assets

Place AI-generated or filmed real-world video clips in this folder.

Required files:

```text
generator-bike-light.mp4
generator-bike-light.jpg
wireless-charging-phone.mp4
wireless-charging-phone.jpg
shake-flashlight-camping.mp4
shake-flashlight-camping.jpg
ebike-speed-sensor-school-gate.mp4
ebike-speed-sensor-school-gate.jpg
```

Recommended export:

- Aspect ratio: 16:9
- Duration: 8-15 seconds
- Resolution: 1280x720 or 1920x1080
- Codec: H.264 MP4
- Audio: optional; the web lesson already provides subtitles
- Avoid visible brand logos, license plates, school names, or identifiable student faces unless you have permission

The app will automatically try to load these files. If a video is missing, it falls back to the built-in SVG life-scene animation.
