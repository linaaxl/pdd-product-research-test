# AI 反馈规则

## 反馈目标

反馈不直接替学生完成推理，而是把学生操作和物理规律连接起来，引导学生观察“磁通量是否变化”“变化快慢”“变化方向”。

## 动态教师提示规则

1. 电动势接近 0
   - 条件：`abs(emf) < 0.05`
   - 提示：让学生先判断磁通量是否变化，再判断有没有感应电流。

2. 电动势较大
   - 条件：`abs(emf) > 2.2`
   - 提示：追问是哪一个参数让磁通量变化率变大。

3. 反向运动
   - 条件：`reverse === true`
   - 提示：要求学生用楞次定律解释电流方向为什么翻转。

4. 匝数较多
   - 条件：`turns >= 120`
   - 提示：强调 e 与 N 成正比，但磁通量本身不因匝数直接变大。

5. 磁场较强
   - 条件：`fieldStrength >= 1.6`
   - 提示：同样位移造成的磁通量变化更明显。

## 参数操作反馈

- 速度为 0：磁铁静止，感应电动势应接近 0。
- 速度增加：运动越快，磁通量变化率越大。
- 匝数改变：匝数越多，同样变化率下感应电动势越大。
- 磁场强度改变：磁通量和感应电动势幅值会同步变化。
- 方向切换：感应电流方向应随磁通量变化率反向。

## AI 追问样例

- 如果只看 e-t 曲线的正负，你能判断磁铁正在靠近还是远离线圈吗？
- 磁通量最大的位置，为什么电动势可能接近 0？
- “阻碍运动”和“阻碍磁通量变化”哪一个表述更准确？
