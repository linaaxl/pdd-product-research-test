# 高二物理 AI 交互课件扩展策略

## 核心判断

不是所有知识点都适合“拖拽仿真 + 曲线 + 问答”这一种模板。真正能提高学生兴趣的课件，必须先回答：

```text
这个知识点能解释什么真实现象？
学生能通过什么操作亲自发现规律？
这个规律能解决什么任务？
```

因此 Harness 后续扩展要从“统一页面模板”升级为“多种课型模板”。

## 五类课件模板

### 1. 现象仿真型

适合：电磁感应、洛伦兹力、带电粒子运动、简谐运动。

特征：

- 有明确可视化对象。
- 学生能拖动、改变参数、观察轨迹或曲线。
- 重点是“看见不可见量”。

### 2. 工程任务型

适合：发电机、变压器、远距离输电、无线充电、电磁炉。

特征：

- 从真实设备或工程问题出发。
- 学生通过调参完成任务，例如“让灯更亮”“减少输电损耗”。
- 重点是“这个知识有什么用”。

### 3. 诊断推理型

适合：楞次定律方向判断、电路故障、图像问题、受力分析。

特征：

- 不一定需要复杂动画。
- 重点是选择、判断、反驳、纠错。
- AI 多角色追问尤其重要。

### 4. 数据实验型

适合：探究加速度、测电阻、验证机械能守恒、研究交流电有效值。

特征：

- 学生采集数据、拟合图像、得出关系。
- 课件重点是数据表、图像、误差和实验报告。

### 5. 场景闯关型

适合：复习课、综合应用、物理竞赛式拓展。

特征：

- 多关卡任务。
- 每关对应一个概念或模型。
- 重点是成就感和迁移。

## 高二扩展建议顺序

| 课题 | 推荐模板 | 兴趣入口 |
| --- | --- | --- |
| 磁铁穿过线圈 | 工程任务型 + 现象仿真型 | 发电机、测速传感器、摇一摇手电筒 |
| 导体棒切割磁感线 | 现象仿真型 | 为什么运动导体能当电源 |
| 楞次定律方向判断 | 诊断推理型 | 反驳“感应电流总是阻碍运动” |
| 自感现象 | 工程任务型 | 为什么开关断开会打火花 |
| 交流发电机 | 工程任务型 | 怎么让交流电更稳定、更强 |
| 正弦交流电 | 现象仿真型 | 发电机转一圈，电压为什么正负交替 |
| 有效值 | 数据实验型 | 为什么 220V 不是峰值 |
| 变压器 | 工程任务型 | 手机充电器为什么能降压 |
| 远距离输电 | 工程任务型 | 为什么高压输电更省能量 |
| 洛伦兹力 | 现象仿真型 | 为什么粒子会拐弯 |
| 质谱仪 | 工程任务型 | 怎么按质量分辨粒子 |
| 回旋加速器 | 场景闯关型 | 怎么让粒子越转越快 |

## Harness 优化方向

后续不应只生成一种 `LessonShell`，而应生成：

- `SimulationLessonShell`
- `EngineeringChallengeShell`
- `DiagnosticDebateShell`
- `DataExperimentShell`
- `QuestReviewShell`

每个新课题先由 Curriculum Agent 判定课型，再让 Interaction Design Agent 选择交互结构。

## 当前电磁感应课件的优化结论

当前课件已经从“公式验证型”升级为“应用任务驱动型”：

- 用发电机解释速度、匝数、磁场强度为什么有用。
- 用测速传感器解释感应电动势脉冲。
- 用摇一摇手电筒解释反向运动和交替电流。
- 用无线充电强调“有磁场不够，磁通量必须变化”。
