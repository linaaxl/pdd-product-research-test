# 交互 Spec：磁铁穿过线圈

## 页面结构

1. 情境引入区：课题、真实应用、实时读数。
2. 交互仿真区：SVG 磁铁、线圈、磁场方向、电流方向。
3. 参数控制区：位置、速度、匝数、磁场强度、反向运动、播放和复位。
4. 实时图像区：Φ-t 曲线、e-t 曲线。
5. AI 追问/教师提示区：基于当前状态生成提示。
6. 学生预测题和即时反馈区：3 道选择题。
7. 课堂挑战任务区：3 个探究任务。

## 核心状态

| 状态 | 类型 | 来源 | 用途 |
| --- | --- | --- | --- |
| `position` | number | 拖拽或滑块 | 计算磁通量、绘制磁铁位置 |
| `speed` | number | 滑块 | 自动运动速度、反馈规则 |
| `turns` | number | 滑块 | 计算感应电动势 |
| `fieldStrength` | number | 滑块 | 计算磁通量和感应电动势 |
| `reverse` | boolean | 开关 | 改变自动运动方向 |
| `playing` | boolean | 按钮 | 控制动画 |
| `dragVelocity` | number | 拖拽估算 | 拖拽时计算感应电动势 |
| `series` | DataPoint[] | 采样 | 绘制曲线 |

## 交互规则

- 拖动磁铁时，课件根据位移和时间估算瞬时速度。
- 播放时，磁铁按 `speed` 和 `reverse` 自动运动。
- 磁铁到达边界后，从另一侧循环进入，便于连续观察曲线。
- 参数改变后必须给出一句短反馈。
- 曲线保留最近采样点，避免投影时过密。

## 多表征同步

- 现象表征：磁铁、线圈、磁场箭头。
- 符号表征：Φ、e、顺/逆时针电流。
- 图像表征：Φ-t、e-t。
- 语言表征：楞次定律解释、教师提示、AI 追问。

## 可访问性与课堂展示

- 字体清晰，按钮和滑块有明确标签。
- 主仿真区域占据首屏核心位置。
- 曲线和读数同屏，适合投影。
- 反馈文本短句化，避免遮挡操作。
