# 高中物理《电磁感应》交互课件原型最终报告

## 已实现功能

- 使用 Vite + React + TypeScript 初始化可运行 Web 课件。
- 使用 SVG 绘制磁铁、线圈、磁场方向、感应电流方向和实时曲线。
- 支持拖动磁铁穿过线圈，并根据拖动速度估算瞬时感应电动势。
- 支持播放/暂停自动运动，支持正向与反向运动。
- 参数控制包含磁铁位置、磁铁运动速度、线圈匝数、磁场强度、是否反向运动。
- 实时显示磁通量 Φ、感应电动势 e、感应电流方向。
- 实时绘制 Φ-t 曲线和 e-t 曲线。
- 包含 AI 追问/教师提示区，提示会随操作状态动态变化。
- 包含 3 道学生预测题，并提供即时反馈。
- 包含 3 个课堂挑战任务。
- 添加基础单元测试，验证感应电动势与速度、匝数、运动方向、静止状态之间的关系。

## 交互说明

- 拖动磁铁：在仿真区按住磁铁左右拖动，观察 Φ-t 与 e-t 曲线的变化。
- 播放运动：点击“播放”，磁铁会按当前速度和方向自动穿过线圈。
- 改变速度：速度越大，e-t 曲线峰值越大；速度为 0 时，感应电动势接近 0。
- 改变匝数：匝数越多，同样运动条件下感应电动势越大。
- 改变磁场强度：磁场越强，磁通量和感应电动势幅值越大。
- 切换反向运动：运动方向反向后，感应电流方向随之反向。

## 教学设计说明

- 情境引入区把发电机、无线充电、电磁炉等真实应用与“变化的磁通量”建立联系。
- 仿真区让学生先观察，再用法拉第电磁感应定律和楞次定律解释现象。
- 参数控制区支持控制变量探究，适合课堂投影演示和学生分组操作。
- 实时图像区把抽象的磁通量变化率转化为可视曲线，帮助学生理解 e 与 Φ-t 曲线斜率的关系。
- 教师提示区根据当前状态动态生成追问，引导学生从“是否变化”“变化快慢”“方向如何阻碍变化”三个层次推理。
- 学生预测题用于即时诊断常见误解，例如把“磁通量大”误认为“一定有感应电动势”。
- 挑战任务用于课堂后半段探究与表达训练。

## 已知问题

- 当前物理模型是课堂可视化近似模型，不是完整三维电磁场数值模拟。
- 电流方向用课件约定的顺/逆时针显示，未区分观察者从线圈哪一侧观察的所有变体。
- 实时曲线保留最近 90 个采样点，刷新页面后不会持久保存实验数据。
